# Brew Heaven UI/UX Design ☕️

A coffee booking mobile app UI/UX design created using Figma. This project focuses on providing a smooth and user-friendly experience for reserving coffee tables and selecting time slots.

## 📱 Features
- Login with mobile number
- OTP Verification
- User Registration (Details form)
- Home screen with coffee options
- Table Reservation system
- Time slot selection
- Booking confirmation screen

## 🎨 Tool Used
- Figma

## 📸 Project Screens
- Login Screen  
- OTP Screen  
- Details Form  
- Home Page  
- Reservation  
- Time Slot Selection  
- Confirmation Screen  

## 🚀 Purpose
This project is created to improve UI/UX design skills and showcase a real-world coffee booking app concept.
